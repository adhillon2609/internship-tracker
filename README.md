A desktop application built with java and javaFX to help students organise and track their internship applications.
Features-
1. Store companies name and their corresponding application start date
2. Mark applications as Applied, rejected or accepted
3. Persistent data storage using local files so application data is available between sessions
4. User-friendly graphical interface built with JavaFX
Technology used-
1. Java
2. JavaFx
3. file handling
4. OOP (object oriented programming)
